# Depose

A GUI version of the board game Coup, made with Python 3.6 and Tkinter.

## Dependencies

* tkinter
* pytest

```bash
apt install python3-tk
pip install pytest
```

## Running

From project root:

```bash
python -m depose.main
```

## Testing

From project root:

```bash
python -m pytest
```
From `tests/`

```bash
cd tests/
pytest
```
